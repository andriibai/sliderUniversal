Setup up your JS

        
            $(document).ready(function () {
                'use strict'
                let params = {
                    id: 'slider-un',
                    activeSlidesLg: 1,
                    activeSlidesMd: 2,
                    activeSlidesSm: 2,
                    activeSlidesXs: 1,
                    slideTime: 3000,
                    slideEffect: 'animate__pulse',
                    slideMargin: 3,
                    slideHeight: 200,
                    autoSlide: false,
                    onHoverStop: true,
                    showArrows: true,
                    showBullets: true
                }

                const slider = new SliderUniversal(params)
                slider.init()
            })

Settings

activeSlidesLg
Shows the number of slides on media screens >= 992
activeSlidesMd
Shows the number of slides on on media screens >= 768
activeSlidesSm
Shows the number of slides on on media screens >= 576
activeSlidesXs
Shows the number of slides on on media screens < 576
slideTime
Autoplay Speed in milliseconds
slideEffect
add effect from library animate.style
slideMargin
Custom margin between images
slideHeight
Custom height slides
autoSlide
Enables Autoplay
onHoverStop
Autoplay Speed in milliseconds
showArrows
Prev/Next Arrows
showBullets
Show bullet indicators
